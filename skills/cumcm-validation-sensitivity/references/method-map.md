# Method Map

| Task | Validation choices |
| --- | --- |
| Prediction | holdout, rolling forecast, residual plot, MAE/RMSE/MAPE |
| Classification | confusion matrix, F1, ROC/AUC, class-wise errors |
| Evaluation | weight sensitivity, rank stability, top-k overlap |
| Optimization | feasibility check, scenario analysis, objective sensitivity |
| Simulation | repeated runs, confidence interval, parameter sweep |
| Network | perturb edges/nodes, compare centrality or path stability |
