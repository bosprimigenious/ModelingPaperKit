# Format Rubric

Check against current official CUMCM rules.

## Page Flow

- Electronic mode: title/abstract first.
- Paper mode: commitment page, numbering page, title/abstract.
- Page numbering restarts at body if required.

## Layout

- A4 paper.
- Margins satisfy official minimums.
- Body font and heading hierarchy are consistent.
- No table of contents unless explicitly required.

## Special Content

- References before appendix.
- AI statement/details in official location.
- Supporting-material list in appendix when needed.
- No identity information in electronic build.

## Build

- `xelatex` path available or missing engine disclosed.
- No unresolved critical LaTeX errors.
- Missing figures and undefined references reported.
