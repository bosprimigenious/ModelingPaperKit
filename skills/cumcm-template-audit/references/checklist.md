# CUMCM Template Audit Checklist

- `main_cumcm.tex` defaults to electronic submission mode.
- Electronic paper starts with title, abstract, and keywords.
- Paper-only commitment and numbering pages are opt-in.
- Page margins satisfy official minimums.
- No table of contents is inserted by default.
- References appear before appendix.
- AI-use statement/details follow current official placement.
- Appendix contains supporting-material file list.
- `git diff --check` passes.
- Build command is attempted or missing TeX engine is reported.
