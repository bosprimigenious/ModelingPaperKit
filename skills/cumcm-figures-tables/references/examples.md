# Example Uses

## Plan Result Visuals

User asks: "每个问题该放哪些图表？"

Do:

- Assign one or more visual artifacts to each subproblem.
- Choose table for precise answers and plot for trends/tradeoffs.
- Suggest captions that state conclusions.

## Audit Existing Figures

User asks: "图表够不够有说服力？"

Do:

- Check references from text.
- Check units, axes, legends, and precision.
- Remove or demote decorative visuals.
