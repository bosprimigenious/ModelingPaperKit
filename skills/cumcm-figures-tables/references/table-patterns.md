# Table Patterns

## Result Table

Columns: subproblem, metric, value, unit, interpretation.

## Parameter Table

Columns: symbol, value, source, meaning.

## Comparison Table

Columns: method, metric 1, metric 2, runtime or complexity, selected reason.

## Sensitivity Table

Columns: parameter, perturbation, output change, conclusion.

Rules:

- Use consistent precision.
- Include units.
- Bold only the key result if needed.
- Do not paste oversized raw data tables into the main text.
