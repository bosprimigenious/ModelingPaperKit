# Official Sources

Use official sources for rule-sensitive decisions.

Primary:

- CUMCM official website: https://www.mcm.edu.cn/
- Current-year competition notices.
- Current-year paper format specification.
- Current-year AI-tool usage rules.
- Current-year registration and submission instructions.

Record:

- URL.
- Page title.
- Publish or update date if visible.
- Access date.
- Exact rule excerpt summarized in your own words.

Do not treat GitHub templates, blogs, or forum answers as authoritative rule sources.
