# Example Uses

## Build A Paper Outline

User asks: "给 2026 国赛 C 题做论文框架。"

Do:

- Map each subproblem to sections.
- Draft abstract result slots, not fake results.
- Create model/result/validation blocks for each task.

## Review A Draft

User asks: "看这篇论文结构哪里弱。"

Do:

- Check problem-answer alignment.
- Find assumptions that are unused.
- Find result sections without quantitative answers.
- Suggest concrete section moves.
