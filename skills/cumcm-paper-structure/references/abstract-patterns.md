# Abstract Patterns

Use a one-page CUMCM abstract as a result-first executive summary.

## Recommended Order

1. Background and task scope: one compact paragraph.
2. Method overview by subproblem: name the model families and why they fit.
3. Key results: include numbers, rankings, thresholds, coordinates, schedules, or decisions.
4. Validation and robustness: mention error metrics, sensitivity, or consistency checks.
5. Conclusion: state the practical answer and model value.

## Sentence Moves

- "针对问题一，本文首先..."
- "为刻画...，建立..."
- "求解结果表明..."
- "敏感性分析显示..."
- "综上，本文模型能够..."

## Avoid

- Empty process claims: "本文进行了深入分析".
- Results without numbers.
- Methods without connection to subproblems.
- Long background that crowds out answers.
