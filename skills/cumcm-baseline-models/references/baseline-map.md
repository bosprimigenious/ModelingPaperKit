# Baseline Map

| Task | Baseline examples |
| --- | --- |
| Prediction | mean/median, linear regression, seasonal naive, ARIMA-lite |
| Classification | majority class, logistic regression, decision tree |
| Evaluation/ranking | normalized weighted sum, entropy weight + TOPSIS |
| Optimization | greedy heuristic, linear programming relaxation, simple enumeration |
| Simulation | deterministic scenario, Monte Carlo with fixed assumptions |
| Network | shortest path, centrality ranking, connected components |

Use stronger models only after the baseline is reproducible.
