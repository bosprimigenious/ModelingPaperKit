# Result Consistency Checklist

- Abstract numbers match result tables.
- Conclusion repeats the final model outputs, not earlier baselines.
- Table captions and body use the same units.
- Figure labels and text use the same variable names.
- Ranking direction is clear: higher is better or lower is better.
- Percent changes and ratios are recomputed from source values.
- Every major claim has a visible source artifact.
