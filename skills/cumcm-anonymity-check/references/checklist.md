# CUMCM Anonymity Check Checklist

Search surfaces:

- `.tex`, `.bib`, `.md`, `.py`, `.ipynb`, `.csv`, `.txt`, `.log`
- generated PDF text and metadata when tools exist
- figure labels and embedded text
- code comments and notebook outputs
- archive filenames and directory names

Patterns:

- school, college, lab, advisor, team member names
- phone numbers, email addresses, student IDs
- local usernames and absolute paths
- EXIF/PDF author metadata
- paper-only commitment fields leaking into electronic PDF
