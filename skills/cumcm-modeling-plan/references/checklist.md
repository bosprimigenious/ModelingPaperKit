# CUMCM Modeling Plan Checklist

- Inputs, outputs, constraints, and evaluation criteria are explicit.
- Candidate methods include simple baseline and stronger model.
- Primary method has a fallback if data or time blocks it.
- Objective function or scoring metric is defined.
- Data preprocessing and feature construction are specified.
- Validation uses holdout, simulation, sensitivity, or consistency checks.
- Expected paper artifacts are listed: equations, algorithm, tables, and figures.
