# Sensitivity Patterns

## One-Factor Sensitivity

Change one key parameter by fixed percentages and observe result changes.

## Scenario Sensitivity

Compare conservative, baseline, and aggressive scenarios.

## Weight Sensitivity

For ranking/evaluation models, perturb weights and check rank stability.

## Noise Sensitivity

Add noise to input data and measure output variance.

## Constraint Sensitivity

Relax or tighten critical constraints and observe feasibility and objective value.

Report the parameter range, response metric, and practical interpretation.
