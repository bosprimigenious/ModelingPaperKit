# Example Uses

## From Problem To Model Plan

User asks: "这题该怎么建模？"

Do:

- Extract inputs, outputs, constraints, and metrics.
- Classify the problem type.
- Propose baseline, primary model, and fallback.
- Define validation and sensitivity plan.

## Compare Candidate Methods

User asks: "随机森林、线性回归、XGBoost 用哪个？"

Do:

- Compare data size, interpretability, accuracy, overfitting risk, and paper explainability.
- Recommend a baseline plus final model.
- Specify figures/tables needed to justify the choice.
