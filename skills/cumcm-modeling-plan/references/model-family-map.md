# Model Family Map

## Prediction

- Baseline: mean, persistence, linear regression.
- Stronger: ARIMA/ETS, random forest, gradient boosting, regularized regression.
- Explainability: feature importance, residual analysis, confidence interval.

## Classification

- Baseline: logistic regression, decision tree.
- Stronger: random forest, SVM, gradient boosting.
- Validation: stratified split, confusion matrix, F1, ROC/AUC when appropriate.

## Optimization

- Linear programming: continuous resource allocation.
- Integer programming: selection, assignment, routing, scheduling.
- Nonlinear programming: physics or nonlinear cost.
- Multi-objective: weighted sum, epsilon constraint, Pareto front.

## Evaluation

- Entropy weight: objective weight from indicator dispersion.
- AHP: expert-judgment hierarchy, use carefully.
- TOPSIS: distance to ideal solution.
- PCA: dimensionality reduction and composite score.
- Grey relational analysis: similarity to reference sequence.

## Simulation

- Monte Carlo: uncertainty propagation.
- Discrete event: queues, arrivals, service.
- System dynamics: feedback loops.
- Cellular automata or agent-based: local rules and spatial interaction.
