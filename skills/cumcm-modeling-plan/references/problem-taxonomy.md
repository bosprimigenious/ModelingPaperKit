# Problem Taxonomy

Use this taxonomy to classify a CUMCM problem before choosing models.

## Prediction

Goal: estimate future or unknown values.
Signals: time series, regression target, forecasting horizon, error metric.

## Classification Or Recognition

Goal: assign labels, detect categories, identify states.
Signals: labeled samples, feature table, confusion matrix, accuracy/F1/AUC.

## Optimization

Goal: choose decisions under constraints.
Signals: resources, costs, schedules, routes, allocations, capacity.

## Evaluation Or Ranking

Goal: compare alternatives under multiple indicators.
Signals: index system, weights, normalized scores, ranking stability.

## Simulation

Goal: reproduce or explore a dynamic process.
Signals: stochastic process, scenario design, agent behavior, repeated trials.

## Network Or Graph

Goal: model relationships, flows, paths, influence, communities.
Signals: nodes, edges, weights, routes, centrality, matching, flow.
