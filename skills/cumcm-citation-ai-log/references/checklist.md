# CUMCM Citation AI Log Checklist

- Each external factual claim, dataset, theorem, algorithm, or software dependency has a citation when needed.
- Web sources include access date when appropriate.
- AI-tool record includes tool name, version/model, provider, use date, purpose, and human verification.
- No-AI statement is present only when true.
- AI-used workflow includes the required supporting detail file.
- References are not invented or padded.
