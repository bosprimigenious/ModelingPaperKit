# Reference Examples

Use the style required by the contest or local template. Examples below are placeholders.

## Journal

```latex
\bibitem{key}
作者.
\emph{题名} [J].
期刊名, 年份, 卷(期): 页码.
```

## Book

```latex
\bibitem{key}
作者.
\emph{书名} [M].
出版地: 出版社, 年份.
```

## Website Or Official Notice

```latex
\bibitem{key}
机构.
页面标题 [EB/OL].
URL, 访问日期: YYYY-MM-DD.
```

## Software Or AI Tool

```latex
\bibitem{key}
工具名称, 版本/模型, 开发机构或公司, 使用日期.
```
