# Final Review Rubric

## Critical

- Build fails and no final PDF exists.
- Electronic PDF contains commitment or numbering pages.
- Any identity leak appears in electronic submission.
- AI-use statement is knowingly false or missing when required.
- A subproblem is unanswered.
- Supporting materials required by the paper are missing.

## Warning

- Abstract lacks quantitative results.
- Model assumptions are unused.
- Important variables are undefined.
- Figure/table is not referenced.
- Validation or sensitivity analysis is weak.
- Code cannot reproduce a major table or figure.

## Info

- Caption can be sharper.
- Table precision can be cleaned.
- Section order can be improved.
- Appendix can better explain support files.

Always state which checks were run and which were blocked.
