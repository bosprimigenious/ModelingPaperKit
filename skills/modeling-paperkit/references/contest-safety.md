# Contest Safety

Use this file when the user is actively participating in a contest, asks about external sources, asks to browse, or asks about AI/source declarations.

## Modes

```text
off
cumcm_active
mcm_active
```

## CUMCM Active

Use extra caution:

- Do not proactively search current problem discussions or solution repositories.
- Do not browse public platforms such as GitHub, CSDN, Zhihu, Xiaohongshu, forums, chats, or social sites for live contest problem discussion.
- Prefer local files and official rules supplied by the user.
- Keep identity and school information out of paper and supporting materials.
- Track AI assistance if AI is used.
- Distinguish electronic submission requirements from full paper/cover-page templates.

## MCM/ICM Active

MCM/ICM allows public inanimate sources but requires citation discipline:

- Log all external sources used.
- Do not ask people outside the team for help.
- Do not post problem content, partial solutions, or methods publicly.
- Track AI model/tool use for the AI use report.
- Keep student, advisor, school, and location information out of the paper.

## General

Never submit files automatically. Provide checklists and warnings; the user is responsible for final submission.
