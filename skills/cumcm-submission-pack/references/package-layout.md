# Submission Package Layout

Confirm current official requirements before final use.

## Electronic Paper

Expected:

- Final PDF.
- Starts with title, abstract, and keywords.
- No commitment page.
- No numbering page.
- No school/member/advisor identity information.

## Paper Version

Expected when paper submission is required:

- Commitment page.
- Numbering page.
- Paper body beginning with title/abstract.
- Identity fields match registration system.

## Supporting Materials

Recommended:

```text
supporting-materials/
  code/
  data-description.txt
  outputs/
  AI 工具使用详情.pdf
  README.txt
```

Only include AI details when applicable under current rules.
