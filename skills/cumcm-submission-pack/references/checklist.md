# CUMCM Submission Pack Checklist

- Final electronic PDF starts at title/abstract page.
- Paper version, if needed, includes commitment and numbering pages.
- Supporting material package contains code, data notes, generated artifacts, and AI-use details if applicable.
- Appendix file list matches package contents.
- Build logs and temporary files are excluded unless intentionally included.
- Identity leaks are checked.
- Final report states which checks were run and which were skipped.
