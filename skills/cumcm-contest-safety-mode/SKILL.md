# CUMCM Contest Safety Mode

Use this skill during the active contest window or whenever the user provides current-year problem content.

## Workflow

1. Determine whether the work is pre-contest preparation or active contest work.
2. If active contest work, avoid browsing public discussions, repositories, forums, social media, or answer-like content about current problems.
3. Keep all external sources and AI assistance logged.
4. Warn before actions that could violate independence, anonymity, or submission rules.
5. Prefer local files, official rules, team-provided data, and deterministic checks.
6. Read `references/checklist.md` for safety review.
7. Read `references/red-lines.md` when deciding whether a requested action is risky.
8. Read `references/examples.md` when responding to borderline requests.

## Guardrails

- Never submit files automatically.
- Do not help copy or conceal external solutions.
- Do not browse live-problem discussions unless the user explicitly instructs and the risk is disclosed.
