# Data Issue Taxonomy

- `critical`: data cannot support the claimed model without correction or disclosure.
- `warning`: data issue can bias results, but a documented mitigation exists.
- `info`: noteworthy data feature that should be mentioned or visualized.

Common issue types:

- missing values
- duplicate records
- inconsistent units
- invalid range
- time-order violation
- category spelling mismatch
- target leakage
- identity or private information
