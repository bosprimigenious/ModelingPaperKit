# Directory Layout

Recommended contest project layout:

```text
paper/
  main.tex
  sections/
  figures/
data/
  raw/
  processed/
code/
  preprocess/
  models/
  analysis/
outputs/
  figures/
  tables/
logs/
README.txt
```

Rules:

- Raw data stays unchanged.
- Processed data is generated by scripts.
- Paper figures are copied or exported from `outputs/figures`.
- Use relative paths only.
- Keep run order documented.
